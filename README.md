# MCP servers

- `packages/mcp-mysql` — MySQL MCP server

## Getting started

```bash
pnpm install-mcp
```

This builds and registers the `local-mysql` MCP server `local-mysql` in

1. Claude Code (~/.claude.json)
2. GitHub Copilot (`~/Library/Application Support/Code/User/mcp.json`).

Start a new session in each tool to pick up the server.

See `packages/mcp-mysql/README.md` for environment variables and manual setup.
